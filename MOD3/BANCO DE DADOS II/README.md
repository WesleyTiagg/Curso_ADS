# 🗄️ Banco de Dados

## 📚 Descrição

Esta pasta reúne atividades práticas realizadas na disciplina de Banco de Dados, utilizando PostgreSQL para criação, manipulação e consulta de dados.

## 🧠 Conteúdos abordados

* DDL (Data Definition Language)
* DML (Data Manipulation Language)
* DCL (Data Control Language)
* Consultas SQL
* Modelagem básica de dados

## 📁 Estrutura

* **ddl/** → Scripts de criação de tabelas e estruturas
* **dml/** → Inserção e manipulação de dados
* **dcl/** → Controle de permissões
* **aulas/** → Scripts desenvolvidos durante as aulas

## 💻 Tecnologias

* PostgreSQL
* SQL

## 📈 Observação

Os arquivos representam exercícios práticos realizados em aula e estudos individuais.
