# Consumo API

Projeto React com Vite para consumir APIs públicas usando Axios.

## Funcionalidades

- Lista de usuários com componente `Card`.
- Exibição de informações completas do usuário em `UserDetailsCard`.
- Galeria de 10 fotos com estado de carregamento e erro.
- Uso de `useState`, `useEffect` e `axios`.

## Como executar

1. Abra o terminal em `consumoAPI`
2. Rode `npm install`
3. Rode `npm run dev`
