# 🎓 Curso de Análise e Desenvolvimento de Sistemas (ADS)

Este repositório reúne minhas atividades, práticas e projetos desenvolvidos ao longo do curso de Análise e Desenvolvimento de Sistemas (ADS), com foco na aplicação prática dos conceitos estudados.

## 📚 Objetivo

Organizar minha evolução técnica de forma estruturada, documentando aprendizados, práticas e projetos reais desenvolvidos durante o curso.

## 🗂️ Estrutura do Repositório

O conteúdo está organizado por módulos e disciplinas:

* **modulo-3/**

  * **banco-de-dados/** → Scripts SQL (DDL, DML, DCL), consultas e práticas em PostgreSQL
  * **programacao-internet/** → Projetos práticos (ex: Pokédex, jogo da velha, biblioteca, etc.)
  * **redes-computadores/** → Comandos e práticas em ambiente de rede
  * **engenharia-software/** → Conteúdos e documentação (em andamento)
  * **estrutura-dados/** → Estruturas e algoritmos (em andamento)
  * **inteligencia-artificial/** → Estudos iniciais e práticas futuras

> ⚠️ Algumas disciplinas ainda estão em desenvolvimento e serão atualizadas conforme o progresso no curso.

## 🚀 Metodologia de Estudo

* Registro contínuo das aulas e atividades práticas
* Organização por módulos e disciplinas
* Versionamento com Git e GitHub
* Melhoria contínua dos códigos e projetos

## 📈 Progresso

Este repositório funciona como um diário de evolução técnica, sendo atualizado frequentemente com novos conteúdos, refatorações e melhorias.
